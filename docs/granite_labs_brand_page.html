Granite Labs uses a dark lab background, granite material forms, isometric technical objects, restrained Electric Cyan accents, and a clean documentation-first layout. The approved website direction is a GitHub-style documentation structure with a horizontal header using the format: [Cube] Granite Labs.

Typography Rules

  • Use Inter for headings, UI labels, and body text.
  • Use JetBrains Mono for code, file paths, settings, and technical callouts.
  • Use bold display headings sparingly for architecture, brand, and key section titles.
  • Keep body text in a neutral system font for dependable rendering.
  • Use accent blue only for links, section headers, and primary emphasis.
  • Prefer sentence case instead of all caps except for small eyebrow labels.
  • Use monospace only for code, color tokens, file paths, and settings names.
