# Blue Iris Installation

This section documents the installation and initial configuration of Blue Iris 6.

## Installation Scope

The installation process includes:

- installing Blue Iris software
- configuring the server environment
- setting up storage locations
- enabling hardware acceleration
- initial system configuration

## Typical Installation Steps

1. Prepare the Windows server or workstation
2. Install Blue Iris software
3. Configure storage folders
4. Configure hardware acceleration
5. Set initial global settings
6. Verify system operation

## Documentation Purpose

This guide will provide a clear, repeatable installation procedure for building a stable Blue Iris system.

Future sections will expand this guide with detailed instructions, screenshots, and recommended settings.
