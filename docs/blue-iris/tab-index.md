# Blue Iris 6 – Tab Index

## 🔹 Overview
This page provides a structured reference to the Blue Iris user interface tabs and where key settings are located.

Use this page as a quick navigation guide when working inside the Blue Iris UI.

---

## 🎯 Objective
- Quickly locate settings in Blue Iris
- Reduce time spent searching menus
- Provide a consistent reference for troubleshooting and setup

---

## 🧠 Key Concepts
- Settings are split between Global and Camera-level tabs
- Some features exist in multiple places
- Profiles and schedules influence behavior across tabs
- Many settings interact (recording, alerts, motion, AI)

---

## 🛠️ Global Tabs

### Main
- Live camera view
- System status
- Quick access to cameras

---

### Status
- Camera health
- Signal and connection status
- Recording indicators

---

### Clips
- Stored recordings
- Event playback
- Clip filtering

---

### Alerts
- Alert history
- Triggered events
- Notification review

---

### Log
- System events
- Errors and warnings
- Debugging information

---

### Settings (Global)
Key areas inside Settings:

#### System
- Startup behavior
- General system settings

#### Web Server
- Remote access
- Ports and connections

#### Users
- User accounts
- Permissions

#### Cameras
- Global camera behavior

#### AI / DeepStack / CodeProject
- AI integration settings

#### Storage
- Folder configuration
- Retention rules

#### Profiles
- Profile definitions

#### Schedule
- Time-based profile switching

---

## 🎥 Camera Tabs

When editing a camera:

### Video
- Resolution
- Frame rate
- Stream configuration

---

### Audio

- Audio stream selection

### Trigger

- Sensitivity and zones
- Trigger timing

---

### Record
- Recording mode
- Pre/post-trigger
- Direct-to-disk

---

### Alerts
- Alert actions
- Notification settings

---

### Schedule
- Per-camera scheduling
- Profile overrides

---

### PTZ / Control
- Camera controls
- Presets and movement
