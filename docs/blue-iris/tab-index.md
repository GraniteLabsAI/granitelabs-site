# Blue Iris – Tab Index

<img src="/img/og/blueiris-system-architecture-og.jpg" style={{width: '100%', borderRadius: '12px', marginTop: '12px', marginBottom: '24px'}} />

## 🔹 Overview

This page provides a structured reference to the Blue Iris user interface tabs and where key settings are located.

Use this page as a quick navigation guide when working inside the Blue Iris UI.

---

## 🎯 Objective

- Quickly locate settings in Blue Iris  
- Reduce time spent searching menus  
- Provide a consistent reference for troubleshooting and setup  

---

## 🧠 Key Concepts

- Settings are split between global and camera-level tabs  
- Some features exist in multiple locations  
- Profiles and schedules influence behavior across tabs  
- Many settings interact (recording, alerts, motion, AI)  

---

- System status  
- Quick access to cameras
