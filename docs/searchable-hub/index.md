## <span class="cube-icon cube-ai"></span> Overview

The GraniteLabs Searchable Hub solves one of the biggest problems in complex systems:

👉 critical settings buried deep with no search.

This system uses AI-powered indexing to extract, organize, and make software interfaces fully searchable.

---

## 🎯 Objective

- Surface hidden settings across complex software  
- Provide fast, searchable access to UI elements  
- Reduce time spent navigating menus  
- Enable structured system documentation  

---

## 🧠 Key Concepts

- AI vision extracts visible UI text  
- Screenshots are converted into searchable data  
- Entire interfaces become indexable  
- Designed for real-world troubleshooting and configuration  

---

## 🛠️ Blue Iris Tab Indexer

A Python-based tool that:

- Captures every tab in Blue Iris 6  
- Extracts text using AI vision  
- Builds a fully searchable index  

### Results

- 47 tabs captured  
- 1,600+ settings indexed  
- Complete UI coverage
