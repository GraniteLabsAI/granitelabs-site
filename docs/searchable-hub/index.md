# GraniteLabs Searchable Hub

One of the most frustrating problems with complex software like Blue Iris is that
critical settings are buried several tabs deep with no built-in search function.
The GraniteLabs Searchable Hub solves this with AI-powered indexing tools built
specifically for the systems documented here.

## Blue Iris Tab Indexer

A Python-based tool that captures every tab in Blue Iris 6 using screenshots,
extracts all visible text using AI vision, and builds a fully searchable HTML index.

A complete run captured 47 tabs and over 1,600 settings and UI elements across
the entire Blue Iris 6 interface.

The live searchable index is available here:

[Open Blue Iris Tab Index](/tools/blueiris-tab-index.html)
