## <span class="cube-icon"></span> Brand Foundation

The GraniteLabs Visual Standard v1 is a locked design system governing all visual output across the site, documentation, and assets.

This page is the authoritative reference for all GraniteLabs design and visual implementation.

---

## 🧠 Core Identity

GraniteLabs is a **technical systems brand**, not a marketing brand.

**Identity:**  
Engineered · Structured · Precise · Modular · Scalable

---

## 🎯 Visual Philosophy

- Built, not designed  
- Clean over decorative  
- Controlled glow over dramatic lighting  
- System consistency over artistic variation  

---

## 🎨 Color System

| Name | Hex | Usage |
|---|---|---|
| Granite Black | `#0B0F14` | Base background |
| Deep Navy | `#0F1720` | Gradient support |
| Slate Blue | `#1B2A3A` | Mid-tone surfaces |
| Steel Blue | `#22384D` | Cube shading |
| Electric Cyan | `#00D4FF` | Primary accent |
| Soft Cyan | `#38BDF8` | Secondary accent |
| Cool White | `#E6EDF3` | Primary text |
| Soft Gray | `#94A3B8` | Secondary text |
| Safety Orange | `#F97316` | Rare alert/accent only |

**Allowed:**  
Monochrome blue/cyan system, neutral stone tones, controlled gradients  

**Forbidden:**  
Purple hues, warm lighting (except alerts), rainbow systems  

---
