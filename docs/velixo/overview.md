# Velixo – Overview

---

## <span class="cube-icon cube-storage"></span> Overview

Velixo provides direct Excel-based access to Acumatica data, enabling flexible and real-time reporting.

It is used for financial reporting, job cost tracking, and operational analysis.

---

## 🎯 Objective

- Enable real-time reporting from Acumatica  
- Provide flexible Excel-based outputs  
- Support job cost and financial analysis  
- Reduce reliance on static reports  

---

## 🧠 Key Concepts

- Live data connection  
- GI (Generic Inquiry) integration  
- Excel formulas linked to Acumatica  
- Report structure and repeatability  
- Data accuracy tied to ERP setup  

---

## 🛠️ Core Uses

### Financial Reporting
- Income statements  
- Balance sheets  
- Cash flow analysis  

---

### Job Cost Reporting
- Project-based cost tracking  
- Budget vs actual analysis  
- Cost category breakdowns  

---

### Operational Reporting
- Customer-level reporting  
- Project summaries  
- Performance tracking

Velixo provides a flexible and powerful reporting layer for Acumatica data.
